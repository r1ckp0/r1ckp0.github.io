Escenas I

My first (and hopefully not the last) work for violin and fixed media. I used SuperCollider for sound generation and Ableton Live only for sequencing. I also made a little MobMuPlat patch (libpd) for the performer to trigger the fixed media clips.

I have used very simple sounds, evoking chip-music and granulated violin samples that I recorded. I won't have a recording until the end of September. The link below contains a zip file with the Score, the fixed media part (not final) and some SC code snippets.
